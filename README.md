# Entities and experiments. Corpus exploration, proto indexing and enrichment workflows

Workshop contribution to the roundtable “Workflows” of the Open Editions Workshop “**Named Entities in Digital Editions. Between Structured Databases and Context-Specific Annotation**” at University of Zurich, 06.–07.03.2026.

A tentative workflow is presented that aims to assist editors in compiling a collection of entities linked to authority records (normdata) efficiently. It uses an LLM to detect entities and suggest possible identifiers, then retrieves data using SPARQL (or other APIs).
